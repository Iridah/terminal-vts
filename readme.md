# VTS - Terminal Management System

Sistema de gestión de inventario basado en terminal (estilo AS/400) para centros de distribución minoristas.

## Características
- **Modo Dual:** Operación offline con base de datos local y soporte para sincronización.
- **Módulo de Valorización:** Cálculo de activos basado en costo neto.
- **Control de Aporte Hogar:** Registro específico para consumo interno.
- **Búsqueda Rápida:** Cruce de datos para consulta inmediata de precios y stock.

## Instalación
Requiere Python 3.x y la librería `pandas`.
